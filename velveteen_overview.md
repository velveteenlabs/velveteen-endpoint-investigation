# Velveteen Framework Overview

Velveteen is a modular endpoint threat detection and analysis framework designed to identify suspicious activity across system processes, persistence mechanisms, and network behavior.

## Core Capabilities

* Process analysis
* Persistence detection
* File artifact inspection
* Network correlation

## Purpose

The framework is designed to:

* Detect suspicious system behavior
* Correlate indicators across multiple layers
* Support structured threat investigations

## Positioning

Velveteen combines elements of:

* Endpoint Detection & Response (EDR)
* Threat hunting workflows
* Forensic analysis
